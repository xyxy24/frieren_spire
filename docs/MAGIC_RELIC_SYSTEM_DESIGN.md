## 1. 参考与改编原则

素材首先来自 B 的《设计稿》，并参考《葬送的芙莉莲》中已经出现的魔法名称和意象。动画官方网站的魔法介绍列出了花田、佐尔特拉克、防御魔法、魔力丝线、驭血、模仿、支配、女神三枪、生活魔法及多种元素魔法，可作为课程原型的参考素材：

- [动画《葬送的芙莉莲》官方魔法介绍](https://frieren-anime.jp/special/magic/)
- [Frieren Wiki 魔法列表与出处索引](https://frieren.fandom.com/wiki/Spells)

游戏效果不机械照搬原作，而是遵守以下规则：

1. 每个普通魔法必须在三槽体系中有明确用途；每个 Boss 魔法必须值得占用唯一的终极槽，不能只有名字不同。
2. 每个遗物至少改变一种决策：选什么魔法、何时施放、如何站位或是否承担风险。
3. 课程版不引入法力值、元素反应网、技能树和随机词缀，避免系统失控。
4. 固定保留三个普通魔法槽，并增加一个只能装备 Boss 魔法的终极槽。终极槽不是通用第四槽，不能消除普通三槽的选择压力。
5. 所有效果必须能通过统一伤害、治疗、状态、冷却和触发管线实现与测试。

## 2. 核心体验

理想的一局不应只是把三个最高伤害魔法塞进槽位，而应出现以下选择：

- 一个短 CD 输出、一个控制、一个保命，形成稳定构筑。
- 两个相互触发的魔法加一个风险型遗物，形成爆发构筑。
- 放弃部分伤害，使用区域、恢复和减速进行阵地战。
- 围绕低生命、冲刺穿敌、连续命中或首次施法形成专门构筑。
- 根据下一层敌人和 Boss 临时换装，而不是一套配置打完整局。

三个普通槽建议分别承担“启动器、核心输出、保险手段”，但不进行强制职业限制；终极槽承担低频、高影响的战况逆转或爆发手段。

### 2.1 Boss 终极槽规则

- 初始为锁定状态，击败 Boss 1 并领取第一本 Boss 魔法后启用。
- 只能装备一个已经获得的 Boss 魔法；Boss 魔法不能进入三个普通槽，普通魔法也不能进入终极槽。
- 所有 Boss 魔法共享同一条 `18 秒` 公共冷却。该状态属于终极槽而不是具体魔法，战斗中换装必须继承剩余冷却。
- 每场战斗开始时终极槽可用；施法成功时启动公共冷却，暂停和打开构筑界面不推进冷却。
- Boss 奖励只加入 Boss 已学列表，不自动替换或装备终极槽。第三位 Boss 的奖励仍可选择和查看，用于保持流程与结算记录一致。
- 终极施法默认参与伤害、治疗、状态和通用倍率结算，但不触发“复制/回响下一次魔法”或“三个普通槽互减冷却”等效果，除非遗物明确写出“Boss 魔法”或“终极槽”。

### 2.2 空间尺度基线

空间设计直接使用当前代码的像素参数，不再引入额外单位：

- 游戏窗口为 `1280×720 px`，当前地面顶边位于 `y=640 px`。
- 主角碰撞体宽 `42 px`、高 `64 px`。
- 普通攻击从角色身体前缘开始，向朝向延伸 `58 px`，判定高度 `36 px`，相对角色顶部向下偏移 `14 px`。
- 主角移动速度为 `260 px/s`，起跳速度 `650 px/s`，重力 `1600 px/s²`；理论跳高约 `132 px`、滞空约 `0.81 秒`、满速跳跃水平距离约 `211 px`。
- “射程”不包含角色自身宽度。角色面向右侧且左边坐标为 `x` 时，普通攻击覆盖 `x+42` 到 `x+100`。
- 投射/直线距离从角色身体前缘开始计算；区域半径从法术中心计算；位移距离按角色碰撞体中心的实际像素移动量计算。
- 无镜头滚动的首版战斗房间保持约 `1280 px` 宽。若后续加入镜头，普通模板可扩展到 `1600～2400 px`，但需要重新验证远程魔法命中率。
- 常规平台水平间隔建议不超过 `170 px`、垂直层差不超过 `100 px`，为跳跃误差保留余量；更大的间隔必须依赖位移、飞行或机关。

距离分档如下：

| 分档 | 距离 | 相对普通攻击 | 设计含义 |
|---|---:|---|---|
| 接触 | `0～70 px` | 与 `58 px` 普攻接近 | 环身覆盖、贴身高伤或防御技能 |
| 近距 | `>70～140 px` | 约 1～2 次普攻长度 | 高风险爆发、斩击、击退 |
| 中距 | `>140～260 px` | 当前多数已实现魔法 | 元素、区域和控制魔法的主要区间 |
| 远距 | `>260～420 px` | 可跨越多个平台 | 投射物、锁定与预判技能 |
| 超远 | `>420 px` | 超过窗口宽度三分之一 | 只用于 Boss 魔法、侦测或长前摇技能 |

## 3. 统一术语与标签

### 3.1 魔法学派

学派用于遗物联动和内容表达，不直接决定伤害类型。

| 学派 | 定位 | 典型魔法 |
|---|---|---|
| 基础术式 | 稳定、通用、短 CD | 佐尔特拉克、多重佐尔特拉克、搜寻魔力痕迹、解除魔法 |
| 元素术式 | 伤害、区域、环境控制 | 火焰、冰箭、石弹、雷电法杖、大地支配 |
| 民间/召唤术式 | 条件式收益与战场工具 | 花田、幻影、石之魔像、飞行魔法 |
| 女神魔法 | 防护、恢复、封锁 | 女神加护、女神三枪、封印魔法 |
| 魔族魔法 | 高上限、控制或代价 | 驭血、支配、模仿、魔力丝线、黄金束缚 |

### 3.2 机制标签

一个魔法可以拥有多个标签。遗物只读取标签，不读取魔法名称。

首版使用：`直接伤害`、`投射物`、`区域`、`持续`、`控制`、`反制`、`标记`、`防御`、`位移`、`恢复`、`召唤`、`血魔法`、`普攻强化`、`Boss魔法`。未实现的屏障、净化、诅咒等标签不得提前作为遗物触发条件。

### 3.3 标准状态

| 状态 | 统一规则 |
|---|---|
| 缓速 | 只降低水平移动速度，不改变攻击前摇；同名缓速取最高值并刷新时间 |
| 寒意 | 冰箭命中时附加 5.5 秒，不直接影响移动；用于让冰箭在自身 4.5 秒基础 CD 恢复后仍可由第二发触发冻结，重复命中刷新时间，火焰魔法命中时清除 |
| 束缚 | 禁止水平移动；黄金束缚对普通敌人为 1.4 秒、Boss 为 0.4 秒，并可被石弹或斩断魔法消费 |
| 硬直 | 中断当前动作；Boss 只接受短硬直，不接受长时间击倒 |
| 冻结 | 中断普通敌人 0.65 秒；由冰箭命中已有寒意或其他减速的目标产生，并可提高浮空坠击伤害 |
| 标记 | 持续期间受到伤害 +12% 并显示攻击范围；由黑冲穿身、幻影破坏、搜寻魔力痕迹或成功解除产生，不会被普通伤害消耗 |
| 封印 | 立即中断目标，并暂时禁止特殊攻击；普通敌人 4 秒、Boss 1.5 秒，已标记目标额外延长 1 秒 |
| 护盾 | 先于 HP 承伤；不同来源护盾相加，但不超过最大 HP 的 30% |
| 加护 | 持续 5 秒，阻止新控制效果并使玩家伤害 +15%；不会移除施放前已经存在的控制 |

## 4. 全局战斗规则

### 4.1 施法事务顺序

每次施法严格按以下顺序结算：

1. 检查槽位、角色状态、冷却和施法条件。
2. 支付 HP 等施法代价；代价不能被增伤、减伤或护盾修改。
3. 启动该魔法冷却。
4. 创建投射物、区域、护盾或状态。
5. 命中时进入统一伤害/治疗管线。
6. 应用控制或负面状态。
7. 触发“施法后”“造成伤害后”“命中后”遗物。
8. 处理击杀；追加伤害不得重新触发自身或同一条触发链。

### 4.2 伤害与冷却边界

```text
最终伤害 = round((基础伤害 × 来源倍率 × 目标倍率) - 固定减伤)
最终冷却 = clamp((基础冷却 + 固定修正) × 冷却倍率, 基础冷却 × 0.5, 基础冷却 × 2.0)
```

- 普通增伤合计上限为 +100%，特殊 Boss 机制可单独突破。
- 受到伤害倍率限制在 50%～200%。
- 单个魔法一次施放对同一敌人只能产生一次“命中”触发；持续伤害的每一跳只算“伤害”，不重复算“施法”。
- 自动施放和回响施放必须标注来源，不能触发另一次自动施放或回响。

### 4.3 控制抗性

- 普通敌人承受 100% 控制时长。
- 精英承受 60%。
- Boss 承受 35%，单次硬控制最长 1 秒。
- 同一目标在 4 秒内再次受到硬控制，持续时间额外减半；4 秒未受控后恢复。
- 支配对 Boss 不改变阵营，只造成 0.8 秒硬直与 4 秒破防。

### 4.4 战斗恢复限制

为了避免玩家留下最后一个敌人无限回血：

- 只有遭遇仍处于战斗状态且至少有一个活敌人时，战斗恢复魔法才生效。
- 每层由魔法产生的 HP 恢复总量不超过最大 HP 的 40%。
- 击败最后一个敌人后立即清除持续治疗区域，但保留掉落物和地图探索。
- 楼梯恢复不计入该上限。

### 4.5 战斗中换装

装备栏仍可随时打开并暂停底层战斗。为了防止轮换全部已学魔法绕过冷却：

- 冷却状态属于“魔法”，不属于槽位；换出后仍继续计时。
- 战斗中换入魔法时，剩余冷却至少为其基础冷却的 50%。
- 同一魔法不能同时装备在多个槽位。
- 战斗外换装不施加 50% 起始冷却。

### 4.6 判定形状与距离规则

- `前方矩形 L×H`：从角色身体前缘沿朝向延伸 `L px`，高度为 `H px`。
- `直线投射 D/尺寸`：最大飞行距离为 `D px`，尺寸表示投射物碰撞盒像素大小。
- `指定点 D，区域 R`：施法点距角色中心最多 `D px`，效果区域半径为 `R px`。
- `自身区域 R`：以角色中心为圆心；平台式实现可使用水平、垂直半径不同的椭圆或 AABB。
- 默认投射物被实心墙体阻挡；只有明确带“贯穿”的魔法例外。
- 默认指定点魔法需要视线；侦测、诅咒和剧情魔法是否忽略墙体必须逐项声明。
- 攻击恰好接触最大距离边界时视为命中；超过边界任意正值即为未命中，便于确定性测试。

## 7. 遗物体系

### 7.1 通用规则

- 每种遗物每局只能获得一次，不设计叠加等级。
- 遗物触发分为：战斗开始、施法、命中、伤害、受击、冲刺穿敌、击杀、楼层结算和经济事件。
- 追加伤害带有 `RelicSecondary` 来源，不再次触发“造成伤害后”的追加伤害遗物。
- 自动施法不启动装备槽冷却，也不触发“首次主动施法”遗物。

## 8. 推荐构筑原型

构筑不是预设职业，而是验证内容之间确实存在多条可行路线。

### 8.1 标记连射

- 魔法：搜寻魔力痕迹、多重佐尔特拉克、解除魔法或封印魔法。
- 遗物：紫葡萄发饰、魔力探测镜片、圣杖之证。
- 节奏：先标记和读招，再用三连光束触发连击；解除成功与标记命中持续返还长 CD。
- 擅长：夸尔、邓肯等前摇明确的施法者，以及阿乌拉召唤出的多目标。
- 弱点：拉奥芬瞬移和鸟形魔物高速位移会降低直线命中率。

### 8.2 花田业火阵地

- 魔法：催生花田、火焰魔法、冰箭或空间破碎。
- 遗物：苍月草的花、钢铁花瓣书签、勇者的花冠。
- 节奏：开场自动花田控速和恢复；需要爆发时主动点燃，再用冰火热冲击或近身范围技收尾。
- 擅长：无法移动的混沌花、追击型无头骑士和冰原狼。
- 弱点：鸟形魔物、拉奥芬以及会主动离开区域的敌人不易长时间留在花田内。

### 8.3 冰火坠击

- 魔法：冰箭、火焰魔法、浮空坠击。
- 遗物：服从天平、古旧魔导书书签、魔力抑制耳坠。
- 节奏：第一次冰箭减速，第二次冻结；根据站位选择火焰触发热冲击，或浮空坠击吃到冻结追加伤害。
- 擅长：高 HP、移动路线稳定的琉格纳、冰原狼和夸尔。
- 弱点：需要两次冰箭建立状态，面对三方向夹击时启动较慢。

### 8.4 冲刺撞墙处刑

- 魔法：魔力打击、黄金束缚、石弹；Boss 魔法可选大致能斩断一切。
- 遗物：克拉フト的护腕、服从天平、破损的高速移动术式。
- 节奏：黑冲穿身完成标记和换边，普通冲刺用于走位；黄金束缚定住后，再用魔力打击或石弹把敌人送向边缘，Boss 斩击可消费束缚追加伤害。
- 擅长：贴身型无头骑士、宝箱怪、冰原狼和红镜龙。
- 弱点：鸟形魔物和远离边界的空中目标难以稳定触发墙击。

### 8.5 驭血爆发

- 魔法：驭血魔法、女神加护、幻影或石之魔像。
- 遗物：暗黑龙的角、魔族金币、镜莲华戒指。
- 节奏：先用诱饵创造安全窗口，再开启加护并以高生命值释放驭血；回响不重复支付 HP。
- 擅长：需要迅速击杀的琉格纳、里希特和 Boss 爆发窗口。
- 弱点：邓肯持续伤害、红镜龙吐焰和多方向小怪会放大低血风险。

### 8.6 高速穿身反击

- 魔法：解除魔法、空间破碎、封印魔法；固有能力以黑冲穿敌换边为核心。
- 遗物：芙拉梅的战术手记、破损的高速移动术式、僧侣的圣典。
- 节奏：读取攻击范围，在攻击生效前用已充能黑冲穿身换边；穿敌伤害和 CD 返还后立刻用解除或空间破碎反打，充能期间用普通冲刺调整站位。
- 擅长：无头骑士、莉涅、夸尔和红镜龙等动作提示清楚的敌人。
- 弱点：鸟形俯冲、拉奥芬换边和邓肯持续龙卷要求不同的闪避判断，不能只依赖固定方向冲刺。

### 8.7 召唤控场

- 魔法：幻影、石之魔像、催生花田；Boss 魔法可选大地支配。
- 遗物：钢铁花瓣书签、圣杖之证、苍月草的花。
- 节奏：用幻影或魔像改变敌人攻击目标，把追击者停留在花田和石柱范围内，再从安全方向输出。
- 擅长：三敌编队、阿乌拉召唤以及冰原狼等强追击单位。
- 弱点：无目标持续区域和全屏类 Boss 攻击不会被诱饵完全解决。

### 8.8 飞行雷杖战法

- 魔法：飞行魔法、雷电法杖、短 CD 伤害魔法。
- 遗物：战士的断斧碎片、克拉フト的护腕、古旧魔导书书签。
- 节奏：飞行避开莉涅落地斩和冰原狼地面猛扑，利用强化普攻返还魔法 CD，落地前释放获得 15% 加成的伤害魔法。
- 擅长：地面攻击占比高的莉涅、冰原狼、里希特石柱。
- 弱点：鸟形魔物、德拉特丝线和邓肯龙卷仍能覆盖空中位置，必须保留冲刺或控制魔法应对。

### 8.9 终极回响

- 普通魔法：女神加护、搜寻魔力痕迹和一个生存工具；Boss 魔法选择破灭之雷、裁决之光、镜像阵或其他终极。
- 遗物：赛丽艾的魔导书残页、镜莲华戒指、芙拉梅的战术手记。
- 节奏：普通槽负责标记、保护和等待窗口，主要爆发来自 15.3 秒终极；镜像阵则强化下一次普通伤害魔法。
- 擅长：Boss 与高 HP 精英。
- 弱点：普通魔法伤害 -8%，终极落空后的真空期明显。

### 8.10 无伤经济流

- 魔法：女神加护、飞行魔法、石之魔像或幻影。
- 遗物：勇者一行的旧铜币、宝箱怪的舌片、一级魔法使徽章；事件可补勇者之剑。
- 节奏：优先保住实际 HP，通过重投和额外商店商品提高后续构筑质量。
- 擅长：熟悉敌人前摇后进行稳定滚雪球。
- 弱点：几乎不直接提高伤害，邓肯持续区域、红镜龙吐焰和拉奥芬偷袭都容易中断无伤奖励。

### 8.11 敌人压力与可用答案

遗物不应成为“遇到某个敌人自动胜利”的钥匙。每种压力至少由基础动作、魔法和遗物三条路径中的两条解决：

| 敌人压力 | 魔法/动作答案 | 遗物支援 |
|---|---|---|
| 宝箱怪咬住、德拉特丝线、混沌花睡眠、阿乌拉支配 | 冲刺、女神加护、解除/封印 | 僧侣圣典、女神石碑碎片 |
| 鸟形魔物俯冲、拉奥芬瞬移换边 | 冲刺、幻影、空间破碎、搜寻魔力痕迹 | 芙拉梅战术手记、破损高速移动术式 |
| 无头骑士、冰原狼、红镜龙的碰撞压力 | 冲刺、飞行、黄金束缚、魔力打击 | 勇者之剑、真·勇者之剑、克拉フト护腕 |
| 阿乌拉召唤和三敌编队 | 贯穿佐尔特拉克、花田、浮空坠击、业火风暴 | 腐败贤老封印碎片、钢铁花瓣书签 |
| 里希特石柱与莉涅落地攻击 | 冲刺、飞行、解除魔法 | 僧侣圣典、芙拉梅战术手记 |
| 邓肯龙卷、红镜龙吐焰的多段持续伤害 | 离开区域、飞行、幻影/魔像争取移动窗口 | 北境御寒斗篷、勇者花冠形成的护盾 |
| 琉格纳与高 HP 敌人的成长型伤害 | 快速爆发、控制、标记增伤 | 暗黑龙角、魔力抑制耳坠、镜莲华戒指 |

## 9. 获取、商店与奖励规则

### 9.1 普通战奖励

- 从普通魔法池提供三个未拥有候选。
- 三个位置固定为一个伤害类、一个控制/反制类、一个全池随机；某一分类耗尽时再按普通池合法候选补位。
- 玩家已有全部普通魔法时改发 15 金币，保持当前原型行为。
- 不引入重复魔法升级，避免第一版同时维护等级和平衡矩阵。

### 9.2 Boss 奖励

- 仅从 Boss 魔法池提供三个未拥有候选。
- 同一次选择中不同时出现功能完全重叠的两个纯伤害大招。
- 第一次领取 Boss 魔法后启用终极槽；获得的 Boss 魔法进入独立列表，不自动装备。
- 第三个 Boss 奖励仍可展示和选择，然后通过楼梯或结算交互进入胜利，保持奖励流程一致。

### 9.3 商店

每个商店保持当前界面的两行五件商品：第一行三本未拥有普通魔法，第二行两件未拥有遗物。遗物候选采用确定性楼层 RNG，并按幕逐步解锁：

- 第一幕：`4001～4012` 首发池。
- 第二幕：在首发池基础上加入 `4013～4020`。
- 第三幕：加入 `4021～4024`，形成完整普通遗物池。
- `4101/4102` 只来自剑之乡事件，不进入商店。

两件遗物中至少一件来自通用/防御/经济类别；另一件可优先匹配玩家已学魔法的标签，但匹配过程仍由种子决定，不能读取玩家即时选择后重新刷新。目标价格为普通魔法 20～35 金币、遗物 35～55 金币；当前原型的 15/20 金币遗物价格只是测试值。幕数加价最多 20%，不根据玩家当前金币动态定价。

### 9.4 事件

- 事件可直接给予扩展魔法、遗物或一次重投机会。
- “奥尔登勋爵的舞会”保留三项：最大 HP +30、随机未拥有魔法、50 金币。
- 生活魔法优先通过事件和商店出现，避免普通战奖励频繁给出暂时无战斗价值的选择。

## 10. 数值预算

### 10.1 普通魔法

| 类型 | 建议范围 |
|---|---|
| 短 CD 直接伤害 | 18～25 伤害，1.2～2 秒 CD |
| 带控制的伤害 | 20～35 伤害，3.5～6 秒 CD |
| 重型爆发 | 38～50 伤害，6～9 秒 CD，带射程/前摇风险 |
| 硬控制 | 普通敌人 1～2.5 秒，8～12 秒 CD |
| 恢复 | 单次 10～20 HP，并受每层 40% 最大 HP 总上限约束 |
| 增伤状态 | +15%～25%，覆盖率不应超过约 50% |

### 10.2 遗物

- 无条件伤害增益通常不超过 15%。达到 20% 时必须附带风险或条件。
- 单件遗物每场战斗的期望追加伤害约为普通敌人最大 HP 的 20%～35%。
- 冷却缩减单次通常为 0.5～1 秒，总下限受基础 CD 50% 限制。
- 遗物不能直接让 Boss 永久失去行动能力。
- 经济遗物应提高选择空间，而不是保证每个商店都能买空。

### 10.3 射程与覆盖预算

普通攻击长度 `58 px` 是所有主动伤害魔法的比较基准：

| 空间优势 | 数值预算修正建议 |
|---|---|
| 射程 `≤100 px` 且不是环身攻击 | 可比同 CD 中距魔法高约 20%～30% 伤害，或获得斩杀/强击退 |
| 射程 `140～260 px` | 使用标准伤害与 CD 预算 |
| 射程 `>260～420 px` | 伤害降低约 10%～20%，或通过细判定、视线、前摇补偿 |
| 射程 `>420 px` | 只用于 Boss 魔法、侦测或长前摇技能，不提供无条件高频输出 |
| 自身区域半径 `≥100 px` | 伤害通常低于同 CD 单体魔法 25%，价值来自多目标和防包围 |
| 指定点区域半径 `≥80 px` | 必须有落点预警、施法距离限制或较长 CD |
| 可穿墙/自动追踪 | 至少附带伤害、速度、目标数量或冷却中的一项代价 |

射程不能单独决定强度，还需同时计算命中宽度、弹速、是否追踪、是否穿透、能否移动施法和垂直覆盖。平衡记录应保存“标准房间内的平均命中率”，不能只比较纸面 DPS。

lineOfSightPolicy, wallCollisionPolicy,

```

```

## 13. 必须覆盖的测试场景

1. 三个普通槽冷却相互独立，普通魔法换装不能刷新冷却；终极槽仍保持独立 18 秒公共 CD。
2. HP 代价先于驭血伤害计算，暗黑龙角和魔族金币只修改伤害、不修改 HP 支付。
3. 苍月草自动花田不占槽、不启动 CD、不触发首次主动施法；仍可被玩家火焰魔法点燃。
4. 镜莲华回响不重复支付 HP、不复制控制、不触发任何施法/命中遗物，也不能再次回响。
5. 圣杖之证每个普通槽每场最多触发一次，始终只减少另外两个普通槽 0.75 秒。

## 14. 需要试玩决定的参数

- 暗黑龙角前 15 秒的风险是否足够抵消全程 +20% 伤害，或应延长到整场战斗。
- 标记增伤 12% 与魔力探测镜片返还 0.35 秒组合后，是否会使标记成为唯一优选路线。
- 僧侣圣典 40% 控制缩短是否会过度削弱混沌花和阿乌拉，必要时可降至 30%。
- 北境御寒斗篷每跳固定减 4 对邓肯与红镜龙是否过强，并确认不同持续区域的来源识别可靠。
- 首发商店遗物价格建议为 35～55 金币；实际价格需与每幕平均收入共同试玩。
- 旧铜币每层 10 金币是否诱导玩家频繁重开追求无伤；若是，改为每幕最多触发两次。
- 赛丽艾残页的普通伤害惩罚采用 -8% 还是 -10%，以终极实际命中率决定。
- 多目标增伤与阿乌拉召唤组合下，诛魔佐尔特拉克是否会使 Boss 战过短。
- 普通攻击 `36 px` 判定高度能否稳定服务雷电法杖、克拉フト护腕和断斧碎片构筑。

这些参数应通过数据配置调整，不应写死在 UI 或敌人代码中。
